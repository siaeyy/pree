## pree

Configurable, procedural tree generation library for Web (canvas).<br>
This library is just for personal use, there wont be much.

#### Why Speedy2D
It is simple, and this is a simple work so all of the other options for rendering are not worth to use.<br>
Besides, I had never used it before, so wanted to use it in this library.

#### Try Example
Simply use `./run_example` script, this requires `wasm-pack` and `python` for building and serving.
